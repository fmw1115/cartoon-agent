# 分镜脚本格式规范

> 适用于AI视频生成工具的画面描述输入

---

## 分镜格式标准

每个分镜包含以下要素：

| 要素 | 说明 | 示例 |
|------|------|------|
| 编号 | 分镜序号 | 01, 02, 03... |
| 时长 | 该镜头持续秒数 | 5秒 |
| 画面 | 画面内容描述 | 教室内，阳光透过窗户... |
| 镜头 | 镜头类型 | 特写/中景/远景 |
| 动作 | 人物动作/表情 | 转头、微笑、叹气 |
| 对白 | 角色对话/旁白 | "今天也要加油啊！" |

---

## 镜头类型定义

| 类型 | 英文 | 用途 | AI提示词示例 |
|------|------|------|-------------|
| 远景 | Wide Shot | 交代环境 | full shot, establishing view |
| 全景 | Full Shot | 展现人物全身 | full body view |
| 中景 | Medium Shot | 人物膝盖以上 | cowboy shot |
| 近景 | Close-up | 人物胸部以上 | close-up |
| 特写 | Extreme Close-up | 脸部细节 | face close-up, detail shot |
| 主观 | POV | 角色视角 | first person view |
| 过肩 | OTS | 越过肩膀拍摄 | over-the-shoulder |

---

## 分镜列表示例

```markdown
| # | 时长 | 画面描述 | 镜头 | 对白/旁白 |
|---|------|----------|------|-----------|
| 01 | 5秒 | 清晨阳光照进卧室，闹钟响起 | 远景→特写 | （闹钟声）啊啊啊！ |
| 02 | 8秒 | 主角睡眼惺忪地坐起来，头发炸毛 | 特写→中景 | 内心OS：五分钟...再五分钟... |
| 03 | 6秒 | 主角看了眼手机，表情瞬间清醒 | 中景→特写 | 内心OS：糟了！要迟到了！ |
```

## AI视频生成提示词技巧

### 画面描述公式
```
[主体] + [动作] + [场景] + [风格] + [光线] + [情绪]
```

### 示例对比

**❌ 模糊描述**
> 一个人在跑

**✅ 清晰描述**
> 日系动漫风格，粉色头发少女在樱花纷飞的校园走廊奔跑，阳光从左侧照射，笑容灿烂，轻风扬起裙摆，铅笔画质感，温暖色调，8fps动画

### 常用风格关键词

| 风格 | 关键词 |
|------|--------|
| 日系动漫 | anime style, Japanese animation, cel shading |
| 写实 | realistic, photorealistic, 3D render |
| 迪士尼 | Disney style, Pixar style |
| 复古 | retro animation, vintage cartoon |
| 像素 | pixel art, 8-bit style |

### 常用情绪关键词

| 情绪 | 关键词 |
|------|--------|
| 开心 | happy, joyful, smiling, laughing |
| 悲伤 | sad, tearful, melancholic |
| 惊讶 | surprised, shocked, amazed |
| 愤怒 | angry, furious, frustrated |
| 害羞 | shy, blushing, embarrassed |

## 运镜方式

| 方式 | 说明 | AI提示词示例 |
|------|------|---------------|
| 推镜 | 逐渐靠近 | slowly zoom in, pushing in |
| 拉镜 | 逐渐远离 | pull back, zooming out |
| 摇镜 | 镜头左右摆动 | pan left, pan right |
| 跟镜 | 跟随主体移动 | following shot, tracking |
| 旋转 | 镜头旋转 | 360 degree rotation, spinning |

## 分镜输出示例（完整版）

```markdown
# 《第一集：迟到的早晨》分镜表

## 分镜1 [5秒]
- **画面**：阳光透过粉色窗帘照进房间，闹钟特写，指针指向7:30
- **镜头**：远景淡入 → 闹钟特写
- **动作**：闹钟震动，床上的人形被子蠕动
- **对白/音效**：闹铃声"滴滴滴滴"

## 分镜2 [8秒]
- **画面**：被子掀开一角，露出炸毛的黑色短发，睡眼惺忪的双眼
- **镜头**：中景
- **动作**：伸手摸索手机，拿起手机屏幕亮起
- **对白**：内心OS"嗯...再睡五分钟..."

## 分镜3 [6秒]
- **画面**：手机屏幕显示"8:15"，主角瞬间清醒的表情变化
- **镜头**：脸部特写
- **动作**：瞪大眼睛，嘴巴张开，慢慢变色
- **对白**：无声表演 + 内心OS"完了完了完了！"

## 分镜4 [10秒]
- **画面**：主角在床上手忙脚乱地换衣服，房间乱糟糟的
- **镜头**：中景 → 近景切换
- **动作**：快速穿衣、找袜子、踩拖鞋
- **对白**：急促呼吸声

## 分镜5 [8秒]
- **画面**：主角冲出门，樱花飘落的街道，拼命奔跑
- **镜头**：远景 + 跟镜
- **动作**：边跑边整理头发，撞到路边的小猫
- **对白**：对小猫喊"对不起！"
```
