# 小说漫剧转换 Issue 模板

---

## Issue 标题

```
[小说漫剧] 《书名》第X章 → AI漫剧剧本
```

---

## Issue 内容

```
## 任务类型
小说文本 → AI漫剧剧本 + 分镜脚本 + 角色卡

## 小说信息
- **书名**：
- **章节**：
- **字数**：
- **核心主题**：

## 漫剧需求
- **目标时长**：1-3分钟
- **情感基调**：搞笑/感人/悬疑/治愈/热血
- **风格**：日漫/国漫/迪士尼/其他
- **目标受众**：儿童/青少年/成人/全年龄

## 小说文本

（粘贴小说章节内容，或描述核心剧情）

---

## 输出要求
1. **剧本**：按场景分幕，包含对白和旁白
2. **分镜脚本**：每个分镜标注画面、镜头、时长、AI视频提示词
3. **角色卡**：核心角色不超过3个，包含外观、性格、口头禅
4. **格式**：Markdown 输出

## 交付标准
- 剧本800-2000字
- 分镜不少于8个
- 每个分镜附带AI视频生成提示词
```

---

## 评论示例（分配给 Agent 后）

```
@hermes 请加载 cartoon-agent skill，按照上面的需求将小说内容转换为漫剧剧本。完成后输出完整的剧本、分镜和角色卡。
```
